# 函数变速音视频播放器

一个基于 TypeScript 的 Web 应用，通过自定义数学函数或图形绘制来控制音视频的播放速度，实现动态变速播放效果。

## 项目用途

本项目允许用户通过以下三种方式定义播放速率函数：

1. **公式输入**：使用数学表达式定义速率函数，支持常用数学函数（sin, cos, log, exp 等）
2. **图形绘制**：在画布上直接绘制函数曲线，支持闭合曲线和方向选择
3. **预设函数**：选择预设的速率模式（恒定、线性加速、正弦波、脉冲波等）

系统会根据定义的函数实时调整音视频的播放速度，并在两个画布上可视化显示：
- **源函数图像**：显示原始函数曲线和当前播放位置
- **播放速率时间曲线**：显示速率随时间的变化轨迹

## 实现方式

### 技术栈

- **TypeScript**：类型安全的 JavaScript 超集
- **HTML5 Canvas**：用于函数图像绘制和可视化
- **Web Audio/Video API**：控制媒体播放速度
- **原生 JavaScript**：无框架依赖，轻量级实现

### 核心功能实现

#### 1. 数学公式解析

- 支持简化的数学语法（如 `sin`, `cos`, `pi`, `^` 等）
- 自动转换为 JavaScript Math 函数
- 变量 `t` 表示当前时间，`d` 表示总时长

#### 2. 图形绘制处理

- **非闭合曲线**：按 X 轴坐标映射到时间，Y 轴映射到播放速率
- **闭合曲线**：使用弧长参数化，支持顺时针/逆时针遍历
- **自动缩放**：将绘制的最小值映射到 1x，最大值映射到最大倍速限制

#### 3. 速率控制

- 使用 `media.playbackRate` API 实时调整播放速度
- 通过 `requestAnimationFrame` 实现平滑的速率更新
- 速率范围限制在 1x 到最大倍速限制之间

#### 4. 可视化

- **源函数画布**：显示原始函数曲线，标记当前播放位置
- **预览画布**：显示速率随时间的变化曲线，包含坐标轴和网格

## 运行方式

### 环境要求

- Node.js（用于 TypeScript 编译）
- 现代浏览器（支持 ES2017+ 和 Canvas API）

### 安装依赖

```bash
npm install
```

### 编译 TypeScript

```bash
# 一次性编译
npm run build

# 或使用监听模式（自动重新编译）
npm run watch
```

### 启动本地服务器

由于浏览器安全限制，需要通过 HTTP 服务器访问，不能直接打开 HTML 文件。

#### 方法一：使用 Python

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### 方法二：使用 Node.js serve

```bash
npx serve .
```

#### 方法三：使用 VS Code Live Server

安装 Live Server 扩展，右键 `index.html` 选择 "Open with Live Server"

### 访问应用

在浏览器中打开：
```
http://localhost:8000
```

### 使用步骤

1. **选择媒体文件**：点击"选择音视频文件"，选择一个音频或视频文件
2. **选择输入方式**：
   - **公式输入**：输入数学表达式，如 `sin((t / d) * pi * 4) + 2`
   - **图形绘制**：在画布上点击或拖拽绘制曲线
   - **预设函数**：从下拉菜单中选择预设模式
3. **调整参数**：设置最大倍速限制（默认 4x）
4. **开始播放**：点击"应用函数并开始播放"按钮
5. **观察效果**：播放速度会根据函数实时变化，画布上会显示当前播放位置

## 注意事项

- 播放速度受浏览器限制，通常范围在 0.25x 到 4x 之间
- 某些浏览器可能对自动播放有限制，需要用户手动点击播放
- 图形绘制模式下，闭合曲线会自动检测并支持循环播放
- 函数值会自动进行归一化处理，确保最小值为 1x
