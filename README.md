# Variable Speed Audio/Video Player with Custom Functions

> 函数变速音视频播放器 | 通过自定义数学函数或图形绘制控制音视频播放速度

[English](README.en.md) | [中文](README.zh.md)

---

## Quick Start

```bash
npm install
npm run build
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

---

## Features

- **Formula Input**: Define rate functions using mathematical expressions
- **Graphical Drawing**: Draw function curves directly on canvas
- **Preset Functions**: Select from predefined rate modes
- **Real-time Visualization**: Dual canvas display of function curves and rate trajectories
- **Closed Curve Support**: Automatic detection and looped playback for closed curves

## Tech Stack

- TypeScript
- HTML5 Canvas
- Web Audio/Video API
- Vanilla JavaScript (no framework dependencies)
